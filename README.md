## Sestavení
**Před spuštěním/sestavením je nutné zaregistrovat si projekt na Firebase.**
### UNIX
1. `git clone https://github.com/Royal-Buccaneers/kodelog --recursive`
2. `./flutterw build web --release`
### Windows
0. Nainstalovat Flutter
1. Clone
2. Build
